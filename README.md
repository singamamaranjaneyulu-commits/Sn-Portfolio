# Sonam Portfolio

This is a portfolio website template built using HTML, CSS, and JavaScript. It showcases various sections such as Home, About, Services, Portfolio, and Contact. The design is modern and responsive, making it suitable for personal or professional use.

## Features

- **Responsive Design**: Adapts to different screen sizes and devices.
- **Smooth Scrolling**: Provides a seamless navigation experience.
- **Lightbox Gallery**: Showcases portfolio images in an elegant lightbox.
- **Contact Form**: Allows visitors to get in touch easily.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/sonamCode/portfolio.git
   ```
2. Open `index.html` in your browser to view the template.

## Customization

Feel free to customize the template by modifying the HTML, CSS, and JavaScript files to suit your needs.
